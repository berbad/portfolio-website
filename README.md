## Portfolio Website

A personal developer portfolio website showcasing projects, skills, and experience. Designed to highlight technical ability and provide a central place for recruiters or collaborators to explore work.

## Features
- Project showcase with descriptions and links
- Skills and technologies section
- About me section
- Contact form
- Responsive design for all devices

## Tech Stack
- Frontend: (React, Next.js)
- Styling: (Tailwind CSS)
- Deployment: (Vercel)
